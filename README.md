# Credit-Card-Fraud-Detection-Using-Machine-Learning
Developing predictive models to detect fraudulent transactions by analyzing customer spending behavior, reducing financial risk, and enhancing transaction security.


## Business Problem
Credit card fraud is a major challenge for financial institutions, leading to significant monetary losses and reduced customer trust. Traditional rule‑based systems often fail to adapt to evolving fraud patterns, making it essential to apply machine learning techniques for proactive detection.

## Project Objectives
- Build predictive models to classify transactions as fraudulent or legitimate
- Handle class imbalance using techniques like SMOTE
- Provide actionable insights to improve fraud detection systems
- Demonstrate how AI can strengthen financial security and customer confidence

## Tech Stack
- Languages & Libraries: Python, Pandas, NumPy, Scikit‑learn
- Algorithms: Logistic Regression, Random Forest, XGBoost, Neural Networks
- Techniques: Feature Engineering, Class Imbalance Handling, Model Evaluation (Precision, Recall, F1‑Score, ROC‑AUC)
- Visualization: Matplotlib, Seaborn

## Expected Outcomes
- Accurate classification of fraudulent vs. legitimate transactions
- Improved detection rates with minimized false positives
